# Python Flask Login Page

This project is a simple login page using a Python Flask backend and a Bootstrap-styled HTML frontend.

## How to Run

1. Activate the virtual environment:
   ```powershell
   .\venv\Scripts\Activate
   ```
2. Start the Flask app:
   ```powershell
   python app.py
   ```
3. Open your browser and go to [http://localhost:5000/login](http://localhost:5000/login)
