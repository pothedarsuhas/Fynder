# Python Flask Login Page

This project is a simple login page using a Python Flask backend and a Bootstrap-styled HTML frontend.

## How to Run

1. Activate the virtual environment:
   ```powershell
   .\venv\Scripts\Activate
   ```
2. Start the Flask app:
   ```powershell
   python app.py
   ```



> **Note:** This is a demonstration only. For production, use secure authentication and session management.
